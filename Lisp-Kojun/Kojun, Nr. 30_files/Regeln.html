Regeln Rules ルール
  1. In jedes Feld eines Gebiets aus N Feldern ist eine Zahl aus dem Bereich 1~N einzutragen.
  2. Jede Zahl muss dabei genau einmal verwendet werden.
  3. Zahlen in orthogonal benachbarten Feldern müssen verschieden sein.
  4. Sind zwei Felder im gleichen Gebiet vertikal benachbart, muss die Zahl im oberen Feld größer sein als die im unteren.
  1. Enter a number in each cell of the grid so that every region of size N contains each number from 1 to N exactly once.
  2. Numbers in orthogonally adjacent cells must be different.
  3. If two cells are vertically adjacent in the same region, the number in the upper cell must be greater than the number in the lower cell.
  1. 太線で区切られた部屋の全てのマスに、1からその部屋のマス数までの数を1つずつ入れます。
  2. 同じ部屋の中でタテに接するマスがある場合、必ず上のマスには下のマスより大きい数字が入ります。おな、同じタテ列にあっても接していなければ、上のマスに小さい数字が入ることもあり得ます。
  3. 同じ数字は、タテヨコに連続しません。
